Todo List (`ipm-todo-list`)

A task management module with local storage persistence.

**Features:**
- Add tasks by typing and pressing Enter
- Click checkbox to mark complete (fades and moves to bottom)
- Right-click to delete tasks
- Scrollable content

**Config:** No configuration needed

```json
{
  "name": "ipm-todo-list",
  "width": 1,
  "height": 2,
  "x": 0,
  "y": 0,
  "config": {}
}
```
